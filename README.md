Arduino Assignment 1
this is my submission for assignment 1 in the 
programming C++ for engineering using arduino
course at Ghana Communication Technology University 

 the program counts down from 5 to 1 using 
 while loops. on each step:
 it prints " count : N " to yhe serial monitor 
 it blinks the on-board LED N times using a function called flashLED()
 when the count down 
 completes it prints " === countdown complete==="

  the concepts demonstrated are 
  variables (int)
  functions with parameters ( flashLED)
  the while loop
  Digital output (digitalWrite, pinMode)
  the serial monitor (Serial begin, Serial.print, Serial. println)

  how to run it 
  1. open the .ino file in the Arduino IDE.
  2. connection an Arduino uno via USB.
  3. select tools> board > Arduino Uno and the correct port
  4. click upload, then open tools> serial monitor (9600)
